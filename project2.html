Sales Dashboard & Insights

Power BI | Excel | DAX

📊 Project Summary

This project involved designing an interactive sales performance dashboard using Power BI. The dashboard offers insights into revenue trends, category-wise sales, and region-wise breakdowns to support better business decisions.

🛠️ Tools & Technologies

Power BI, Microsoft Excel, DAX, Power Query

💡 Key Highlights

  • Designed an interactive dashboard with filters, slicers, and dynamic visuals.
  • Cleaned and transformed raw Excel data using Power Query.
  • Created DAX measures to compute KPIs such as Total Revenue, Profit %, and YoY Growth.
  • Enabled drill-down capability by product, category, and regional segmentation.

📌 Outcome

The dashboard helped stakeholders track key metrics, analyze underperforming categories, and identify top-selling regions, supporting data-backed decisions in a visually rich format.
