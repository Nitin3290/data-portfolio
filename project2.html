AI-Based Virtual Clothes Try-On

Computer Vision | GAN | Flask

📌 Project Summary

This project enables users to virtually try on clothes by uploading a personal image and a clothing item. The system overlays the clothing onto the user image using a deep learning pipeline involving segmentation, pose estimation, and generative adversarial networks (GANs).

🛠️ Tools & Technologies

U-2-Net, OpenPose, GANs, Flask, Python

💡 Key Highlights

  • Preprocessed user and clothing images using pose estimation and image segmentation.
  • Integrated a GAN pipeline to generate realistic try-on results.
  • Deployed the try-on interface with a clean Flask-powered web interface.
  • Tested the pipeline on varied user inputs and clothing types for generalizability.

📌 Outcome

The tool provides a user-friendly interface for virtual try-on, enhancing online shopping experiences and personal styling previews. It demonstrates how AI can revolutionize retail and personalization in fashion.
