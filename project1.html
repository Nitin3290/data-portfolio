Retinal Image Analysis

Diabetic Retinopathy Detection using CNN

🧠 Project Summary

This deep learning project focused on building a computer vision pipeline to detect early-stage diabetic retinopathy from retinal scans. Using a convolutional neural network (CNN), the system learns to differentiate healthy eyes from ones showing signs of damage.

🛠️ Tools & Technologies

Python, TensorFlow, OpenCV, Jupyter Notebook

⚙️ What I Did

  • Built a CNN model to classify retinal images with an accuracy of 92%.
  • Preprocessed images using OpenCV including resizing, histogram equalization, and image sharpening.
  • Applied data augmentation techniques to improve model generalization.
  • Visualized classification results using confusion matrices and precision-recall metrics.

📊 Outcome

This solution demonstrates the effectiveness of deep learning in medical diagnostics and highlights how computer vision can assist doctors in detecting diabetic retinopathy early and accurately.
